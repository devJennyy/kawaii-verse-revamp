## <a name="tech-stack">⚙️ Tech Stack </a>

- React.js
- TailwindCSS
- TypeScript
- ShadCN
- Framer Motion
- SwiperJS
- Jikan API
- Axios


## <a name="quick-start">⚡ Commands </a>

1. Install dependencies:

```bash
npm i
```

2. Start the development server:

```bash
npm run dev
```
